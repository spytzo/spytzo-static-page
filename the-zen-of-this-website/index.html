The Zen of this website

(Inspired by The Zen of Python)

  1. Web is rich. Websites should reflect that.
  2. You can be rich without being flashy. Remove the unnecessary.
  3. Web should be fast and mobile-first. PageSpeed matters.
  4. Privacy is important. Don’t sell your user data.
  5. Web should be accessible.
  6. Web is social.
  7. Web is written. Typography matters.
  8. Web should be crawlable. Use Schema.
  9. Don’t be afraid to use updated technology. Don’t get starstruck by it.
